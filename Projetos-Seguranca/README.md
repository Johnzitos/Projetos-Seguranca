# Portfólio de Projetos - Engenharia de Segurança da Informação

Repositório central para ferramentas de segurança ofensiva e defensiva, scripts de automação e implementações de compliance. O objetivo deste portfólio é demonstrar competências práticas em Python e Bash Script aplicadas a cenários de Blue Team e Red Team.

## Projetos

| 02 | [Network Port Scanner](./Projeto-02-PortScanner) | Recon / Red Team | Python, Socket | Em Desenvolvimento |

## Tecnologias e Competências
* **Linguagens:** Python 3, Bash.
* **Normas:** Implementações baseadas em NIST SP 800-63B e ISO/IEC 27001.
* **Conceitos:** Entropia da Informação (Shannon), Protocolos TCP/IP, Expressões Regulares.
